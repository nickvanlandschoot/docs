## Conceptual Guidance

Corporate events follow the M&A lifecycle:

- **deal_type** categorizes the transaction structure
- **regulatory_status** tracks approval process
- **deal_value_usd** enables financial threshold filtering

**Key Distinction**: "Friendly Acquisition" vs "Hostile Takeover" - determined by target company board approval.
