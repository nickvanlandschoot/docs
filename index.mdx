## Welcome to Intrace

Intrace provides comprehensive event data through the Corpus Product API, covering multiple domains including conflict, cyber, crime, corporate, political, climate, organized crime, and information operations.

## Quick Start

1. **Get your API key** - Contact your Intrace representative
2. **Read the [Integration Guide](/guides/corpus-api/integration-guide)** - Complete setup instructions
3. **Explore the [Event Taxonomy](/guides/corpus-api/taxonomy)** - Understand available event types
4. **Check the [API Reference](/api-reference/corpus/events/list-events)** - Detailed endpoint documentation

## What's Available

The Intrace Corpus Product API provides access to:

- **Events** - Query and filter events across multiple categories
- **Actors** - Discover actors and their relationships
- **Locations** - Access country and region data
- **Metadata** - Explore available categories, types, and subtypes

All data is accessible via RESTful API endpoints with comprehensive filtering, pagination, and export capabilities.
